# Taller de Programación Competitiva 1 - 2025-2

Este repositorio contiene el material y ejercicios del curso **Taller de Programación Competitiva 1**, dictado en el Semestre 2 de 2025.

##  Estructura del Repositorio

- **`Material/`**: Contiene las diapositivas de las clases y apuntes teóricos (PDFs).
- **`Tareas/`**: Directorio para las entregas de tareas y ejercicios prácticos.
- **`Syllabus...pdf`**: Planificación oficial del curso.

##  Recursos

- [Codeforces](https://codeforces.com/)
- [UVA Online Judge](https://onlinejudge.org/)
- [CSES Problem Set](https://cses.fi/problemset/)
- [VJudge](https://vjudge.net/)
